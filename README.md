# Proyek Metode Eliminasi Gauss (Python)
Nama Proyek: Kalkulator Gauss 

Proyek ini merupakan Aplikasi penerapan Matematika Lanjutan dengan materi Eliminasi Gauss berbasis CLI (Command Line) 

## Tujuan membuat proyek: 
1. Sebagai syarat penilaian MID pada LAB-AP
2. Mempermudah user dalam memperoleh Himpunan penyelesaian dari SPL (Sistem Persamaan Linear) dalam matriks augmented persegi menggunakan eliminasi gauss dengan efektif dan akurat.

## Fitur yang disediakan:
- Input ordo matriks
- Input koefisien dan hasil
- Normalisasi matriks
- Eliminasi Gauss 
- Subtitusi Balik
- Tampilkan langkah proses
- Analisis hasil 
- Penyimpanan riwayat input dan output jika bernilai benar secara otomatis pada file (hasil.txt)

## Cara Menjalankan
1. Install NumPy 
2. Jalankan `python main.py`
3. Input ordo
4. Input koefisien
5. Hasilnya akan muncul lengkap dengan matriks augmented
