# Proyek Metode Eliminasi Gauss (Python)

Proyek ini menerapkan konsep matematika lanjutan (Eliminasi Gauss) menggunakan Python.
Fitur:
- Input sistem persamaan linear
- Normalisasi matriks
- Eliminasi Gauss 
- Back Substitution
- Tampilkan langkah proses
- Penyimpanan hasil ke riwayat
- Analisis hasil 

## Cara Menjalankan
1. Jalankan `python main.py`
2. Ikuti menu interaktif
